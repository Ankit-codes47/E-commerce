# 🛒 EasyCart – Online Grocery Store

EasyCart is a responsive online grocery shopping website designed to provide a simple and user-friendly shopping experience. Users can browse grocery products, add items to their cart, manage delivery addresses, view orders, and track order status.

The website includes responsive Login and Signup pages and is designed to work smoothly across desktop, tablet, and mobile devices.

## ✨ Key Features

- 🛍️ Browse 30+ real-life grocery products
- 🛒 Add to Cart and manage product quantities
- 🔍 Search and filter products by category
- 📦 Order tracking system
- 📋 My Orders page
- 📍 Delivery address management
- 👤 Responsive Login and Signup pages
- ⭐ Customer reviews
- ❓ FAQ page
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage for cart data

## 🎯 Project Purpose

This project was created to practice front-end web development and understand the basic structure and functionality of a real-world e-commerce grocery website.
